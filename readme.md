# Студкод 2026

Интерактивная платформа для проведения хакатона «Студкод» в колледже КИГМ №23. Проект объединяет в себе информационный лендинг, систему подачи заявок и панель управления для жюри.

## 🚀 Основные возможности

- **Информационный хаб**: Главная страница с описанием хакатона и информацией о призах.
- **Брифы заданий**: Четыре ключевых направления разработки (Навигация, Общение, Рейтинг, Поддержка).
- **Система регистрации**: Форма отправки проектов с автоматической генерацией ID заявки и синхронизацией с облачной базой данных.
- **Панель управления (Admin)**: Защищенный интерфейс для жюри, позволяющий просматривать все входящие заявки, переходить по ссылкам на репозитории и удалять записи.
- **Современный UI**: Адаптивный «стеклянный» дизайн (Glassmorphism) с глубокими цветами и анимациями.

## 🛠 Технологический стек

- **Frontend**: HTML5, CSS3 (Tailwind CSS).
- **Icons**: FontAwesome.
- **Backend/Database**: Firebase Firestore (Real-time updates).
- **Auth**: Firebase Anonymous Authentication.
- **Fonts**: Inter & JetBrains Mono.

## 📦 Структура данных (Firestore)

Проект использует строго заданную структуру путей для обеспечения безопасности и корректной работы прав доступа:
`artifacts/studcode-2026/public/data/submissions`

Каждый документ заявки содержит:

- `team`: Название команды.
- `group`: Учебная группа.
- `captain`: ФИО капитана.
- `git`: Ссылка на репозиторий.
- `demo`: Ссылка на демо/дизайн.
- `ts`: Таймштамп создания.
- `userId`: ID пользователя.

## ⚙️ Установка и запуск

Поскольку проект реализован в формате Single File Application (все ресурсы в одном HTML-файле), установка не требуется:

1. Скачайте файл `index.html`.
2. Откройте его в любом современном браузере.
3. Для работы функционала отправки и администрирования требуется подключение к интернету (для связи с Firebase).

Или откройте по ссылке: [stud-code.ru]

## 🔐 Доступ администратора

Для входа в панель управления:

1. Добавьте к URL параметр `?admin=kigm23`
   Пример: `https://stud-code.ru?admin=kigm23`
2. Перейдите на вкладку "Админ" в навигации.
